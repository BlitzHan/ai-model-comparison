# AI Model Karşılaştırma - Dinamik Dashboard

Bu proje, Prof. Dr. Oğuz Ergin'in [ai-model-timeline](https://github.com/prof-oguzergin/ai-model-timeline) projesindeki verileri temel alarak geliştirilmiş, dinamik bir karşılaştırma platformudur.

## Özellikler
- **Dinamik Model Seçimi:** Karşılaştırmak istediğiniz modelleri manuel olarak seçebilirsiniz.
- **Otomatik Satır Filtreleme:** Seçilen modellerde veri bulunmayan benchmark satırları otomatik olarak gizlenir.
- **En İyi Değer Vurgulama:** Her satırda seçili modeller arasındaki en yüksek başarım puanı otomatik olarak vurgulanır.
- **Modern Arayüz:** Karanlık mod destekli, responsive ve şık dashboard tasarımı.

## Atıf
Bu çalışma, Prof. Dr. Oğuz Ergin'in değerli verileri ve orijinal çalışması üzerine inşa edilmiştir. Kendisine bu verileri serbestçe kullanma izni verdiği için teşekkür ederiz.

## Nasıl Kullanılır?
1. Dosyaları indirin.
2. `index.html` dosyasını tarayıcınızda açın.
3. Veya GitHub'a yükleyip **GitHub Pages** özelliğini aktif ederek yayına alın.
